Quitting will write the final test report and exit TEST MODE. The following will happen:

  • Bob and Karen fixtures dropped
  • Production users (Darrin, Codex) restored

Reversible: you can re-enter test mode at any time, but a new session ID will be generated. Production data is never affected.
