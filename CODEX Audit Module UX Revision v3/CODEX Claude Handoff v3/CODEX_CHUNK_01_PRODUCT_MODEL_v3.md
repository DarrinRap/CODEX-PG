# CODEX Chunk 01 - Product Model v3

The Audit Module is an intelligent inbox for Panda Gallery testing feedback. Screenshots, region captures, transcript text, test results, and metadata arrive through Dropbox. The module checks completeness, analyzes the package, groups findings, ranks urgency, drafts a sender response, prepares Claude Code tasks, and later verifies fixes.

It should feel automated but not reckless. Analysis and drafting are automatic. Sending responses, assigning code work, and closing findings require human approval.
