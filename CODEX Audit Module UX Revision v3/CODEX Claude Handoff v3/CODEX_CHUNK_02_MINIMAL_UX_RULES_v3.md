# CODEX Chunk 02 - Minimal UX Rules v3

The surface should show only the main decision at each step. Use Lightroom/Photoshop-style collapsible panels for advanced details.

Visible by default: Review findings, Approve finding, Edit wording, Approve response, Create Claude Code package, Verify fixed.

Collapsed by default: raw JSON, Dropbox sync logs, hashes, duplicate candidates, model diagnostics, archive metadata, full transcript, delivery metadata.

One primary action per screen. At most three visible secondary actions.
