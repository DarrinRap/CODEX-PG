Quick reference

HELP
The ? circle, top right. Opens a key reference.
PROGRESS
"Step X of Y" text, top left.
TITLE
The orange heading of the step.
INSTRUCTIONS
The plain text block that tells you what to do. Scrolls if long.
QUESTION LIST
The rows with colored dots. The active one has an orange left stripe.
SUMMARY STRIP
The thin "X pass · X fail · X skip" line that shows up after you answer one question.
ACTION BAR
The 6 buttons at the very bottom: Cancel, Pause, Back, PASS, FAIL, SKIP.
FAIL PANEL
The red-bordered textbox that opens when you press FAIL.
