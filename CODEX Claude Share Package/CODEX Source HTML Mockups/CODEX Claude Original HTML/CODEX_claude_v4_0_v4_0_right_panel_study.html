Already in LightroomSlider
Fine-step (arrow keys), coarse-step (shift+arrow), scroll wheel adjust, inline numeric editing, double-click label to reset, drag tooltip, dragStarted / dragEnded for two-tier rendering, disabled-state palette.
Worth adding
Center detent + bidirectional fill for bipolar sliders — ~10 lines in paintEvent. Track tick at center, fill draws from center outward. Hover scale on thumb — 15% larger on hover, 0.12s transition.
Skip (diminishing returns)
Histogram brush-over on the track, per-channel color tints on Temp/Tint sliders (LR does this with gradient backgrounds) — nice but adds rendering cost and visual noise. Dental radiographs don't need temperature sliders anyway.
Dental-specific
Detent positions besides zero — e.g., a detent at +30 contrast (your Auto Enhance baseline for radiographs, per _auto_enhance). Lets users "feel" the default as they drag.

Things worth noticing

  • › Meta text beside section title: "off" for Tone Curve when zeroed, "5 saved" for Presets, "auto" for auto-detected radiograph. Quiet, monospace, text-dim. Communicates state without forcing the section open.
  • › Subsection headers (Tone, Presence, Sharpening, Noise Reduction) are even quieter than main sections — plain gray text, no triangle, no toggle. Pure hierarchy marker.
  • › "Auto" next to Tone is a subtle text link, not a button — matches LR exactly. Runs auto-tone for that subsection without triggering a full Auto Enhance.
  • › Bottom action row (Copy / Paste / Previous / Reset) is flat, equal weight, with Reset styled destructive. Pasted from LR convention; works regardless of which sections are open.
  • › Transform section adds a "Fine" subsection with a bipolar rotation slider — deliberate radiograph alignment (±5° in 0.1° increments, useful for straightening tilted PAs).

4 · Summary — what to build, in priority order

Everything marked add in the table above, grouped by effort vs value. Everything marked have is working today; everything marked defer sits in v4.2+ alongside the customization plan.

Ship with v4.0 shell (small, high-value)
  • › Bipolar slider center detent + fill-from-center — visual polish only, ~10 lines in LightroomSlider.paintEvent
  • › Collapsible sections with persistence — wrap existing widgets in a section container, one QSetting per section
  • › Section meta text ("off", "5 saved", "auto") — computed from current state, purely visual
  • › Image metadata row under histogram — read existing fields from DB record
  • › Fine rotation slider (±5°, 0.1° step) — extends existing rotation code
v4.1 features (medium effort, real workflow value)
  • › User-saved adjustment presets section — customization plan 4.1, top Tier 3 item
  • › Named history snapshots — LR-parity feature, critical for consult before/after
  • › Clarity / local contrast slider — real radiograph readability improvement
  • › Detail subsection — sharpening amount/radius + noise reduction
  • › Per-image tags / clinical notes field in Info tab
  • › Image category dropdown (inline re-classify)
  • › Bone-detail / CLAHE checkboxes — stage toggles for transparency
Defer to v4.2+ (customization plan territory)
  • › Interactive Tone Curve editor — power-user, 10% of dentists
  • › Annotation stamps / templates (customization plan 4.7)
  • › Layer groups / named layer sets
  • › EXIF / sensor metadata dump
