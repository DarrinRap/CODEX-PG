Implementation notes

Phase 1 ships State 1 → State 2 → State 3, plus the file-write to disk. No toast, no review, no session-integration — just the capture loop. ~250 LOC in new region_capture.py + ~20 LOC in panda_gallery.py for the shortcut + menu + handler.

Phase 2 adds the toast and review dialog (~80 LOC).

Phase 3 wires append_manual_screenshot + _active_step_n + step_state_changed fire (~30 LOC across results_writer.py and instruction_pane.py).

Phase 4 is the pane test plan and manual verification.

All four phases ship as separate vX.YZ commits so regressions are bisectable.
