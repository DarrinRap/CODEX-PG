Instruction Pane — redesign v1

Target file: instruction_pane.py · Closes: #107 (two-row footer waste) · #108 (oversized FAIL textarea) · #110 (in-row PASS/FAIL/SKIP scroll out of view) · #111 (cramped/not modern)

Design principles

  • Action bar is always reachable. PASS / FAIL / SKIP / Next live at the pane bottom, sibling to Cancel / Pause / Back. They never scroll with content.
  • Breathing room. Root padding 20/22, section gap 14, button height 32, all uniform. No squished corners.
  • FAIL textarea is capped. min 64, max 84, internal scroll on overflow. Never dominates the pane.
  • One content area per step kind. Single-expected shows body + EXPECTED + filler. Checklist shows body + scrolling list + summary strip. Mutually exclusive.
  • Summary strip is subtle but present. Right-aligned, muted text, one line above the action bar. Counts use the existing dot colors.
  • No new colors. Everything pulled from styles.py.
