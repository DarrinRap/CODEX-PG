PANDAGallery — Toolbars & Panels

Every clickable control in the shell — top toolbar, tool strip, right-panel tabs

Top toolbar

Persistent strip below the menu bar. Always visible. Canvas-bound actions are disabled when no image is loaded.
OpenOpen a single image file (Ctrl+O) — quick edit without a patient association.
Undo / RedoPer-image edit history (Ctrl+Z / Ctrl+Y). Separate from Back / Forward navigation.
Zoom+ / Zoom−Step zoom (Ctrl+= / Ctrl+−). Mouse wheel also zooms on the canvas directly.
FitFit image to viewport (Home). First thing you reach for after loading an image.
ExportExport with annotations baked in. PNG / JPEG / TIFF / BMP.
PrintPrint preview dialog — lets user pick printer, paper size, fit-to-page.

Tool strip · drawing & annotation

Vertical column on the left of the canvas. Visible only in loupe (single-image) mode. Single-key shortcuts match Lightroom / Photoshop convention.

V · Select — move, resize, and re-select existing annotations. Default tool.

B · Brush — freehand vector strokes. Adjustable width and color in the Drawing panel.

R · Rectangle · E · Ellipse · A · Arrow — shape annotations with resize handles. Hold Shift to constrain (square / circle).

T · Text — editable text annotation; double-click to edit in place.

C · Crop — non-destructive crop. Can be reset any time.

I · Invert — one-shot toggle (not a persistent tool). Returns to Select after firing.

M · Measure is proposed for v4.1 per UX design doc §3.6 (calibrated radiograph ruler).

Right-panel tabs · properties & editing

Six tabs swap what the right panel shows. Adjust / Drawing / Layers / History are gated behind "image loaded." Series shows only when viewing an arrangement.
InfoFilename, date captured, dimensions, file size, image type, category. Shows for library selection or loaded image.
AdjustBrightness · Contrast · Sharpness · Saturation sliders. Auto-enhance button. Copy / Paste / Reset adjustments. Adaptive — context-appropriate controls for radiograph vs photo.
DrawingBrush width, color swatches, fill / stroke options for shape tools. Auto-activates when you pick a drawing tool.
LayersVisibility toggles, z-order, delete. Every annotation is a layer.
HistoryUndo stack visualized. Click any step to jump back. Named snapshots (proposed). Scoped per-image.
SeriesShown only in Arrange module. Template info, slot count, Mount All / Clear All, Edit in Designer, Export / Print.
